# Fundamentos de Java para Principiantes
## Resumen del Curso
Este curso de 1 hora introduce a estudiantes de secundaria a los fundamentos de la programación con Java, incluyendo
programación orientada a objetos. 

**¡No requiere descargas!** Solo abre tu navegador web.

**Duración total**: ~60 minutos

**Público Objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación

**Prerrequisitos**: Ninguno

**Herramientas necesarias**: **¡Solo tu navegador web!** (Chrome, Firefox, Safari, Edge)

**📂 Repositorio de Código Fuente**: [https://github.com/BL-App-Open-Source-1ASI0729-2610-12029/-java-fundamentals-course-Veltrix.git](https://github.com/BL-App-Open-Source-1ASI0729-2610-12029/-java-fundamentals-course-Veltrix.git)

# Secuencia de la Lección

## Lección 1: Introducción a la Programación y a Java (10 minutos con 40 segundos)
- **Descripción**: En esta lección aprenderás qué es la programación y cómo las computadoras siguen instrucciones para realizar tareas. También conocerás Java, uno de los lenguajes de programación más utilizados en el mundo, y crearás tus primeros programas mostrando mensajes en pantalla y realizando operaciones sencillas.
- **Enlace**: [Ver la lección](https://youtu.be/WNL-p4ycs1M?si=g_dewgIH5bJ2Q-ox)
- **Conclusiones clave**: 

* Programar consiste en darle instrucciones a una computadora para resolver problemas o realizar tareas.
* Java es un lenguaje de programación ampliamente utilizado en aplicaciones, sistemas empresariales y desarrollo de software.
* Todo programa Java sigue una estructura básica para poder ejecutarse.
* Las instrucciones permiten mostrar información y realizar operaciones simples.

- **Práctica**: [Clic para programar](https://www.jdoodle.com/ia/1Ubf) **¡Inicio inmediato!**

## Lección 2: Variables, Tipos de Datos y Entrada por Consola (12 minutos con 20 segundos)
- **Descripción**: En esta lección se presenta cómo un programa en Java puede trabajar con información para volverse interactivo y útil. A lo largo de la lección, se explica de manera sencilla cómo almacenar datos, diferenciar entre información numérica y textual, y permitir que el usuario ingrese información desde la consola.
- **Enlace**: [Ver la lección](https://youtu.be/617ouxxTnGE)
- **Conclusiones clave**: 
* Los programas pueden guardar información para utilizarla más adelante, lo que permite crear aplicaciones más dinámicas.
* La interacción hace que los programas sean más útiles y cercanos a situaciones reales
* El aprendizaje de estos conceptos permite pasar de programas estáticos a programas interactivos.
* Es posible aprender programación utilizando únicamente herramientas en línea, de forma accesible y práctica.

- **Práctica**: [Clic para programar](https://www.jdoodle.com/ia/1Ubr) **¡Inicio inmediato!**

## **Lección 5:** Clases y Objetos en Java

- **Descripción**: En esta lección aprenderás uno de los conceptos más importantes de la programación orientada a objetos: las clases y los objetos. Entenderás la diferencia entre el molde y el ejemplar, cómo definir atributos que guardan los datos de cada objeto, cómo usar un constructor para darle vida a tus objetos y cómo agregar métodos que definan sus acciones. Al finalizar, habrás creado tu propio programa con múltiples objetos funcionando juntos.
- **Enlace**: https://youtu.be/9IGuCRdQZEo
